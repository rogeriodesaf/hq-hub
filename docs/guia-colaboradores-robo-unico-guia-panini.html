Guia do colaborador: robo unico Guia + Panini

Como gerar um JSON pronto para importar no HQ-HUB usando PowerShell.

Objetivo

Este guia mostra como rodar um unico robo que faz tres coisas de uma vez:

  1. Pega os dados das edicoes no Guia dos Quadrinhos.
  2. Pega as capas na Panini usando uma sequencia de URLs.
  3. Gera um JSON final pronto para colar/importar no HQ-HUB.
Importante: nao cole as URLs soltas no PowerShell. As URLs entram dentro de um comando completo.

Antes de comecar

  • Use um computador com o projeto HQ-HUB baixado.
  • Abra o PowerShell dentro da pasta do projeto.
  • Copie e cole um comando por vez.
  • Espere o comando terminar antes de fazer o proximo passo.

Passo 1: abrir o PowerShell no lugar certo

Abra a pasta do projeto:

C:\Users\User\OneDrive\Area de Trabalho\HQ-HUB

Clique na barra de endereco da pasta, digite powershell e aperte Enter.

Depois confirme o local com:

pwd

O resultado deve mostrar a pasta HQ-HUB.

Passo 2: confirmar o Python

No PowerShell, rode:

python --version

Se aparecer uma versao do Python, pode seguir.

Passo 3: testar se o robo responde

Este comando nao gera arquivo. Ele apenas mostra a ajuda do robo.

python docs/importacao/ferramentas/fluxo-essencial-hqhub/robo_gerar_importacao_guia_panini.py --help

Se aparecer a lista de opcoes do robo, esta tudo certo.

Passo 4: separar as informacoes da colecao

Antes de rodar o comando final, tenha estas informacoes:

Campo O que colocar Exemplo
URL do Guia Preferencialmente a pagina de capas da colecao. https://www.guiadosquadrinhos.com/capas/saga-do-lanterna-verde-a/sa011149
URL inicial da Panini A pagina da primeira edicao na sequencia da Panini. https://panini.com.br/a-saga-do-lanterna-verde-01
Quantidade Quantidade de edicoes que serao geradas. 10
Titulo da serie Nome principal da colecao no HQ-HUB. Saga do Lanterna Verde, A
Fase Temporada, fase ou serie. Primeira temporada
Volume Numero usado para diferenciar fases com o mesmo titulo. 1

Passo 5: rodar o robo

Exemplo completo para A Saga do Lanterna Verde:

python docs/importacao/ferramentas/fluxo-essencial-hqhub/robo_gerar_importacao_guia_panini.py `
  --url-guia "https://www.guiadosquadrinhos.com/capas/saga-do-lanterna-verde-a/sa011149" `
  --url-panini-inicial "https://panini.com.br/a-saga-do-lanterna-verde-01" `
  --quantidade 10 `
  --saida "docs/importacao/rascunhos/saga-do-lanterna-verde/saga-do-lanterna-verde-1-temporada-com-capas-panini.json" `
  --titulo-serie "Saga do Lanterna Verde, A" `
  --fase "Primeira temporada" `
  --editora Panini `
  --volume 1

Depois de colar o comando, aperte Enter e aguarde terminar.

O que deve aparecer

Durante a execucao, o PowerShell deve mostrar mensagens parecidas com:

URL inicial do Guia: ...
Executando:
Processando 1/10: ...
Arquivo gerado: ...
Capas encontradas: 10
Capas aplicadas: 10
Pronto.
Se aparecer Pronto e Avisos: 0, o arquivo final foi gerado com sucesso.

Arquivos gerados

O robo gera um arquivo final e tambem arquivos intermediarios.

Arquivo Para que serve
*-base-guia.json JSON base gerado com dados do Guia dos Quadrinhos.
*-capas-panini.txt Lista de capas encontradas na Panini, uma por linha.
*-com-capas-panini.json JSON final, pronto para importar no HQ-HUB.

No exemplo, o arquivo final fica em:

docs/importacao/rascunhos/saga-do-lanterna-verde/saga-do-lanterna-verde-1-temporada-com-capas-panini.json

Passo 6: importar no HQ-HUB

  1. Abra o sistema HQ-HUB.
  2. Entre em Importacao.
  3. Clique em Selecionar JSON.
  4. Escolha o arquivo final gerado pelo robo.
  5. Revise o conteudo em Conteudo JSON.
  6. Clique em Importar para o catalogo.

Quando o arquivo ficar grande demais

Se o HQ-HUB mostrar erro interno do servidor ao importar um JSON muito grande, gere lotes menores.

A quantidade de edicoes geradas e controlada pelo parametro --quantidade.

Exemplo: para gerar apenas as 3 primeiras edicoes:

--quantidade 3

O comando completo ficaria assim:

python docs/importacao/ferramentas/fluxo-essencial-hqhub/robo_gerar_importacao_guia_panini.py `
  --url-guia "https://www.guiadosquadrinhos.com/capas/saga-do-lanterna-verde-a/sa011149" `
  --url-panini-inicial "https://panini.com.br/a-saga-do-lanterna-verde-01" `
  --quantidade 3 `
  --saida "docs/importacao/rascunhos/saga-do-lanterna-verde/saga-do-lanterna-verde-1-a-3-com-capas-panini.json" `
  --titulo-serie "Saga do Lanterna Verde, A" `
  --fase "Primeira temporada" `
  --editora Panini `
  --volume 1

Para gerar as proximas edicoes, comece pela URL da edicao seguinte no Guia e pela URL Panini correspondente.

Regra: o campo --url-guia deve ser a URL do Guia da primeira edicao do lote. O campo --url-panini-inicial deve ser a URL da Panini da mesma edicao.

Exemplo para continuar pela edicao 4:

--url-guia "URL_DA_EDICAO_4_NO_GUIA"
--url-panini-inicial "https://panini.com.br/a-saga-do-lanterna-verde-04"
--quantidade 3
--saida "docs/importacao/rascunhos/saga-do-lanterna-verde/saga-do-lanterna-verde-4-a-6-com-capas-panini.json"

Se o lote for da edicao 4 ate a 10, use a URL do Guia da edicao 4, a URL Panini da edicao 4 e --quantidade 7.

Recomendacao: quando a importacao grande falhar, use lotes de 3 a 5 edicoes por arquivo.

Como trocar para outra colecao

No comando, normalmente voce troca apenas estes trechos:

  • --url-guia
  • --url-panini-inicial
  • --quantidade
  • --saida
  • --titulo-serie
  • --fase
  • --volume

Erros comuns

Erro: o PowerShell diz que nao encontrou o arquivo do robo.
Solucao: confirme se voce esta dentro da pasta HQ-HUB usando o comando pwd.
Erro: quantidade de capas diferente da quantidade de edicoes.
Solucao: confira se a URL inicial da Panini e a quantidade estao corretas.
Erro: pagina da Panini inexistente.
Solucao: abra a URL inicial da Panini no navegador antes de rodar o comando.

Regra de ouro

Primeiro teste a URL da Panini no navegador. Se a primeira URL abrir corretamente, o robo geralmente consegue seguir a sequencia.
