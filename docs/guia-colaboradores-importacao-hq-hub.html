Guia rapido para colaboradores do HQ-HUB

Como editar edicoes, preencher o cadastro visual do JSON e importar historias para o catalogo.

1. Encontrar a edicao no catalogo

  1. Abra o catalogo interno do HQ-HUB.
  2. No campo de busca, pesquise o titulo que deseja editar.
  3. Clique na serie interna correspondente.
  4. Ao lado, confira as edicoes daquele titulo.
  5. Clique em Ver detalhes na edicao que deseja editar.

2. Editar campos basicos e capa

Na tela de detalhes da edicao, revise e edite os campos basicos: numero, titulo, data, formato, paginas, preco, descricao e URL da capa.

Importante: nunca use URL direta da imagem do Guia dos Quadrinhos no campo de capa. O Guia costuma bloquear acesso externo a imagens. Prefira URLs de capa da Panini, Amazon ou outro site confiavel que permita carregamento fora da pagina original.

Depois de editar, clique em atualizar/salvar. Se a edicao nao tiver historias ou publicacoes originais para cadastrar, isso ja resolve a manutencao basica.

3. Quando usar Importacao

Use a area Importacao quando uma edicao tiver historias internas, creditos, publicacoes originais ou varias historias que precisam ser vinculadas ao catalogo.

O formulario visual gera um JSON no formato que o HQ-HUB entende. JSON significa JavaScript Object Notation.

4. Preencher dados da serie

origem.url: URL da fonte consultada. Pode ser Amazon, Panini, Guia dos Quadrinhos ou outra fonte. Pode ficar em branco quando nao houver fonte.

serieBrasileira.titulo: nome da serie no Brasil.

serieBrasileira.fase: fase editorial, se existir.

serieBrasileira.editora: editora brasileira, por exemplo Panini.

serieBrasileira.volume: volume ou temporada da serie.

O campo volume e muito importante quando existe mais de uma fase com o mesmo titulo. Exemplo: uma saga pode ter V1 e V2. Isso evita misturar edicoes de temporadas diferentes.

totalEdicoes e totalHistorias sao calculados automaticamente pelo formulario visual.

5. Preencher a edicao

Para cada edicao, preencha os campos disponiveis:

  • numero da edicao;
  • data de publicacao e texto publicado;
  • editora e licenciador;
  • categoria, genero e status;
  • numero de paginas, formato e preco de capa;
  • URL da capa;
  • descricao ou sinopse.

Se for cadastrar mais de uma edicao da mesma serie de uma vez, use + Edicao.

6. Preencher historias

Se a edicao tiver mais de uma historia, preencha a primeira e clique em + Historia para adicionar as demais.

Para cada historia, informe sempre que possivel:

  • ordem;
  • titulo em portugues;
  • titulo original, se existir;
  • quantidade de paginas;
  • resumo, creditos e observacoes;
  • serie original;
  • numero original;
  • ano original;
  • texto da publicacao original.

Para o sistema criar o vinculo de historia, os campos essenciais sao titulo em portugues, serie original e numero original.

7. Gerar e importar o JSON

  1. Clique em Atualizar JSON pelo formulario.
  2. Revise o JSON gerado no campo de texto.
  3. Clique em Importar para o catalogo.
  4. Aguarde o processamento.
  5. Confira o resumo de edicoes criadas, edicoes atualizadas, historias criadas e vinculos criados.

Quando o processamento termina sem erro, a edicao fica incluida ou atualizada no catalogo do HQ-HUB.

Checklist rapido

  • Pesquise antes para evitar duplicidade.
  • Confira se a serie correta e o volume correto foram usados.
  • Use capa de Panini, Amazon ou fonte que permita carregamento externo.
  • Evite imagem direta do Guia dos Quadrinhos para capa.
  • Use Importacao quando houver historias/publicacoes originais.
  • Atualize o JSON pelo formulario antes de importar.
