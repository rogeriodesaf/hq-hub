Guia muito simples: Robos de importacao e capas

HQ-HUB - Passo a passo para qualquer colaborador executar sem duvidas.

Objetivo

  1. Gerar um JSON de importacao a partir de uma URL.
  2. Aplicar capas no campo urlCapa usando um arquivo TXT.
  3. Ter um JSON final pronto para importar no sistema.

Antes de comecar

  • Abra o PowerShell.
  • Copie e cole um bloco de comando por vez.
  • Espere terminar antes de ir para o proximo bloco.

Passo 1: Entrar na pasta do projeto

Set-Location "C:\Desenvolvimento\hqhub\hq-hub"

Passo 2: Ativar o ambiente Python

.\.venv\Scripts\Activate.ps1
Se aparecer bloqueio de script, rode este comando e tente de novo: 
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

Passo 3: Gerar JSON base a partir da URL

Este comando pega a URL da colecao, segue as edicoes e gera um arquivo JSON base.

c:/Desenvolvimento/hqhub/hq-hub/.venv/Scripts/python.exe \
  docs/importacao/ferramentas/robo_importador_texto.py \
  --url "http://www.guiadosquadrinhos.com/edicao/saga-do-batman-a-1-serie-n-1/sa011126/157954" \
  --seguir-galeria \
  --saida "docs/importacao/rascunhos/a-saga-do-batman/saga-base.json" \
  --titulo-serie "A Saga do Batman" \
  --editora "Panini" \
  --volume 1

Passo 4: Aplicar capas no JSON (urlCapa)

Este comando pega o TXT com uma capa por linha e preenche o campo urlCapa em ordem.

c:/Desenvolvimento/hqhub/hq-hub/.venv/Scripts/python.exe \
  docs/importacao/ferramentas/robo_aplicar_url_capa.py \
  --entrada "docs/importacao/rascunhos/a-saga-do-batman/saga-base.json" \
  --urls "docs/importacao/rascunhos/a-saga-do-batman/capas-amazon-1-a-36-playwright.txt" \
  --saida "docs/importacao/rascunhos/a-saga-do-batman/saga-com-capas.json"

Passo 5: Conferir se deu certo

$j = Get-Content "docs/importacao/rascunhos/a-saga-do-batman/saga-com-capas.json" -Raw | ConvertFrom-Json
"totalEdicoes: $($j.edicoes.Count)"
"comCapa: $((@($j.edicoes | Where-Object { -not [string]::IsNullOrWhiteSpace($_.urlCapa) })).Count)"
"semCapa: $((@($j.edicoes | Where-Object { [string]::IsNullOrWhiteSpace($_.urlCapa) })).Count)"

Opcional: modo estrito (para validar quantidade)

Se quiser forcar erro quando a quantidade de URLs nao bater com a quantidade de edicoes, use --estrito:

c:/Desenvolvimento/hqhub/hq-hub/.venv/Scripts/python.exe \
  docs/importacao/ferramentas/robo_aplicar_url_capa.py \
  --entrada "docs/importacao/rascunhos/a-saga-do-batman/saga-base.json" \
  --urls "docs/importacao/rascunhos/a-saga-do-batman/capas-amazon-1-a-36-playwright.txt" \
  --saida "docs/importacao/rascunhos/a-saga-do-batman/saga-com-capas.json" \
  --estrito

Erros comuns e solucao rapida

Erro: "Nao reconhece o comando"
Solucao: confirme se esta em C:\Desenvolvimento\hqhub\hq-hub e se a venv esta ativada.
Erro: "Impossivel conectar-se ao servidor remoto"
Solucao: backend nao esta rodando. Suba o backend antes de importar pela API.
Erro: falta ou sobra capa
Solucao: confira o TXT. Deve ter uma URL por linha, na ordem das edicoes.
