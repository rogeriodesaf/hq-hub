# Backlog

## Usuários e Autenticação

- Como usuário, quero criar uma conta para acessar minha coleção.
- Como usuário, quero fazer login para acessar dados protegidos.
- Como usuário, quero consultar meus dados básicos.

## Catálogo

- Como usuário, quero cadastrar editoras.
- Como usuário, quero cadastrar séries.
- Como usuário, quero cadastrar edições.
- Como usuário, quero cadastrar criadores.
- Como usuário, quero vincular criadores a edições.
- Como usuário, quero consultar publicações de um criador em ordem cronológica.
- Como usuário, quero relacionar uma série com outra quando houver continuação, reboot ou relançamento.

## Coleção

- Como usuário, quero adicionar uma edição à minha coleção.
- Como usuário, quero listar minha coleção.
- Como usuário, quero consultar edições faltantes de uma série.
- Como usuário, quero ver um resumo da minha coleção.
- Como usuário, quero organizar compras planejadas por mês.
- Como usuário, quero visualizar minha coleção como uma estante.

## Amizades

- Como usuário, quero enviar solicitação de amizade para outro colecionador.
- Como usuário, quero aceitar ou recusar solicitações de amizade.
- Como usuário, quero remover alguém da minha lista de amigos.
- Como usuário, quero bloquear outro usuário.

## Visibilidade da Coleção

- Como usuário, quero definir se minha coleção é privada, visível para amigos ou pública.
- Como usuário, quero permitir que meus amigos vejam minha coleção.
- Como usuário, quero visualizar a coleção de um amigo quando ele permitir.

## Classificados

- Como usuário, quero marcar uma HQ da minha coleção como disponível para venda.
- Como usuário, quero marcar uma HQ da minha coleção como disponível para troca.
- Como usuário, quero informar preço, estado de conservação e descrição do anúncio.
- Como usuário interessado, quero ver quais colecionadores estão vendendo ou trocando determinada edição.
- Como usuário interessado, quero entrar em contato com o anunciante por WhatsApp.
- Como anunciante, quero pausar ou encerrar meu anúncio.

## Moderação e Confiança

- Como usuário, quero denunciar um anúncio inadequado.
- Como usuário, quero denunciar outro usuário.
- Como sistema, quero permitir ocultar contato quando o anunciante não desejar exibir WhatsApp.
- Como sistema, quero exibir aviso de responsabilidade nos anúncios.
