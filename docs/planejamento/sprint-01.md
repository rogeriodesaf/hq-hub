# Sprint 01

## Objetivo

Criar a base técnica inicial do backend do HQ-HUB, com cadastro, consulta e autenticação de usuários.

## Escopo

- Configurar projeto Quarkus.
- Configurar PostgreSQL e Flyway.
- Criar cadastro de usuários.
- Criar consulta de usuários.
- Criar autenticação com JWT.
- Criar tratamento global de exceções.
- Criar documentação inicial de endpoints.

## Fora do Escopo

- Comunidade.
- Solicitações de amizade.
- Compartilhamento de coleção.
- Classificados de venda e troca.
- Pagamentos.
- Carrinho de compras.
- Frete.
- Comissões.
- Intermediação financeira.

## Observação

Comunidade e classificados estão planejados para fases futuras e não fazem parte do MVP técnico inicial. A Sprint 01 deve continuar focada em usuários e base do backend.
