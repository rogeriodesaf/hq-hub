# Aviso de Responsabilidade

O HQ-HUB não intermedeia pagamentos, entregas, trocas ou negociações. Os anúncios funcionam como classificados entre colecionadores. Toda negociação deve ser combinada diretamente entre os usuários, por canais externos, como WhatsApp. O sistema apenas organiza e divulga os anúncios.

## O Que o HQ-HUB Faz

- Permite divulgar HQs disponíveis para venda.
- Permite divulgar HQs disponíveis para troca.
- Permite vincular anúncios a itens da coleção.
- Permite informar dados úteis do anúncio, como preço, estado de conservação, cidade e forma de contato.
- Pode gerar um link de contato externo, como WhatsApp, quando o anunciante permitir.

## O Que o HQ-HUB Não Faz

- Não processa pagamentos.
- Não possui carrinho de compras.
- Não calcula frete.
- Não cobra comissão sobre venda.
- Não intermedeia entrega.
- Não garante conclusão da negociação.
- Não atua como marketplace nesta fase.

## Diretriz de Produto

Toda tela, endpoint ou mensagem futura relacionada a anúncios deve deixar claro que a negociação é externa e de responsabilidade dos usuários envolvidos.
